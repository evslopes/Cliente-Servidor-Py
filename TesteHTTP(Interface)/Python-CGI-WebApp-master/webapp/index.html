About this WebApp

This WebApp uses Python 3 and CGI (Common Gateway Interface)

Python Scripts read the public key content from a text file and generate HTML code to display it as a webpage

The server that is handling this webpage is written in Python and is CGI enabled to run the scripts correctly

Working

generate.py imports keyreader and template_engine modules

keyreader.py converts text data into a list and displays each line in HTML format

template_engine.py generates the entire HTML code
